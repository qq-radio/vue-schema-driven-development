# BasicTime 时间

## 简介

- 提供统一的时间格式化组件

## 基本使用

:::demo
basic-time/basic
:::

## BasicTime Attributes

| 属性名   | 说明             | 类型                           | 默认值       | 是否必填 |
| -------- | ---------------- | ------------------------------ | ------------ | -------- |
| `time`   | 要格式化的时间值 | `string` \| `number` \| `Date` | -            | -        |
| `format` | 时间格式化的格式 | `string`                       | `YYYY-MM-DD` | -        |
