# BasicCopy 复制

## 简介

- 提供统一的复制组件

## 基本使用

:::demo
basic-copy/basic
:::

## BasicCopy Attributes

| 属性名 | 说明             | 类型     | 默认值 | 是否必填 |
| ------ | ---------------- | -------- | ------ | -------- |
| `text` | 要复制的文本内容 | `string` | -      | -        |

## BasicCopy Slots

| 插槽名    | 说明       | 插槽参数类型 |
| --------- | ---------- | ------------ |
| `default` | 自定义内容 | -            |
